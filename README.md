# Gentoo Linux overlay

## Generate checksums for new ebuild

1. Download archive (use SRC_URI)
2. Rename archive
3. Run `./manifest-generator.sh PATH_TO_FILE/file.rpm PATH_TO_MANIFEST/Manifest`
